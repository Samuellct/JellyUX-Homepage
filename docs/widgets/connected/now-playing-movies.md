# Now Playing

**Category:** Connected (TMDb)

Displays movies currently playing in theatres, from TMDb, optionally filtered to a configured region,
that are present in your local library.

## Parameters

Shares only the [common parameters](../../configuration.md#common-per-widget-parameters). Refresh
frequency, page count, and the region filter are configured globally in the
[TMDb Cache settings](../../configuration.md#tmdb-cache).
