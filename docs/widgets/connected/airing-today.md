# On TV

**Category:** Connected (TMDb)

Displays TV shows currently on the air (from TMDb's `tv/on_the_air` endpoint, which includes any show
with an episode airing within the next 7 days) that are present in your local library.

## Parameters

Shares only the [common parameters](../../configuration.md#common-per-widget-parameters). Refresh
frequency and page count are configured globally in the
[TMDb Cache settings](../../configuration.md#tmdb-cache).
