# Top Rated Movies

**Category:** Connected (TMDb)

Displays all-time top rated movies from TMDb (filtered by a minimum vote count, so obscure titles with
a handful of 10/10 votes don't crowd out well-established favorites) that are present in your local
library.

## Parameters

Shares only the [common parameters](../../configuration.md#common-per-widget-parameters). Refresh
frequency, page count, and the minimum vote count threshold are configured globally in the
[TMDb Cache settings](../../configuration.md#tmdb-cache) (the "Min votes" field is shared with
[Top Rated Shows](top-rated-shows.md)).
