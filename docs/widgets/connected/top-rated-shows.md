# Top Rated Shows

**Category:** Connected (TMDb)

Displays all-time top rated TV shows from TMDb (filtered by a minimum vote count, so obscure titles
with a handful of 10/10 votes don't crowd out well-established favorites) that are present in your
local library.

## Parameters

Shares only the [common parameters](../../configuration.md#common-per-widget-parameters). Refresh
frequency and page count are configured globally in the
[TMDb Cache settings](../../configuration.md#tmdb-cache). The minimum vote count threshold is a single
field shared with [Top Rated Movies](top-rated-movies.md), found on that widget's status row.
