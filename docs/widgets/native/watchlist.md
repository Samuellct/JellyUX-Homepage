# Watchlist

**Category:** Native

The user's Watchlist -- items marked via the bookmark toggle on card overlays, the item detail page,
or the "More" menu (see the Watchlist tab). Reuses the exact same `Likes`-based query as the
Watchlist tab itself, so the two always stay in sync.

## Parameters

Shares only the [common parameters](../../configuration.md#common-per-widget-parameters). No
widget-specific settings.
