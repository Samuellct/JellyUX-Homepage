# Director

**Category:** Admin

Displays items directed by a specific director chosen by the administrator. You can add multiple
Director sections, each pinned to a different director.

## Parameters

In addition to the [common parameters](../../configuration.md#common-per-widget-parameters):

| Field | Description |
|---|---|
| Value | The director this section shows, picked from an autocomplete list of people credited as directors in your library. |
