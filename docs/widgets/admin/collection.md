# Collection

**Category:** Admin

Displays items belonging to a specific collection (BoxSet) chosen by the administrator. You can add
multiple Collection sections, each pinned to a different collection.

## Parameters

In addition to the [common parameters](../../configuration.md#common-per-widget-parameters):

| Field | Description |
|---|---|
| Value | The collection this section shows, picked from an autocomplete list of collections present in your library. |
