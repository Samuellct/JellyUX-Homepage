JellyUX Homepage

    General

    Show JellyUX widgets on the home screen. Disable to revert to the native Jellyfin home screen.

    API Keys

    Required for Connected widgets that use The Movie Database. Leave empty if not needed. Accepted formats: 32-character hexadecimal key (v3), or v4 bearer token starting with "ey".
    Invalid TMDb key format. Expected: 32 hexadecimal characters, a v4 bearer token, or leave empty.

    Cache

    How long the home screen widget layout is cached per user. Reduce to see configuration changes more quickly; increase to lower server load.
    How often TMDb data is refreshed. Increase to reduce external API calls; decrease for fresher content.

    TMDb Cache

    Status of the cached TMDb data used by Connected widgets.

    Each page is about 20 items from TMDb, so "Pages: 3" fetches roughly 60 items for that list.

    Widgets

    Use the arrows to change the display order. Changes take effect after saving.
